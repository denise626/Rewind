The Code for REWIND


//Get the string from the page
//controller function
function getValue(){

    document.getElementById("alert").classList.add("invisible");
    
    let userString = document.getElementById("userString").value;
    
    let revString = reverseString(userString);
    
    displayString(revString);
}


//Reverse the string
//logic function
function reverseString(userString){

    let revString = [];

    //last position in an array is name.length - 1

    //reverse a string using a for loop
    for (let index = userString.length - 1; index >= 0; index--) {
        revString += userString[index];
        
    }

//revString will add to itself on every iteration of this loop
    
    return revString;
}


//Display the message with the reversed string to the user
//view function

function displayString(revString){

    //write to the page
    document.getElementById("msg").innerHTML = `Your string reversed is: ${revString}`;
    
    //show the alert box
    document.getElementById("alert").classList.remove("invisible");
}

The Code for Rewind is structured in three functions.

Rewind Functions

The function getValue is our controller function. It allows us to get the value from userString that the user has entered. When invisible is added to the alert classList, the alert will be made invisible until you get down to the displayString function. The userString is only available in this block of code. By using reverseString(userString), it can then be passed to the next block of code in a function. The displayString(revString) is used to display the reverseString.

The function reverseString is our logic function. It allows us to get the value that the user typed in and we are able to spell the string backwards. We declare an array in this function and run a for loop that decrements instead of increments so we can go from the end of the string to the beginning. By using let userString.length - 1, we start at the last position of the array. Index >=0 allows us to count backwards and we end up stopping when the index = 0. The index -- creates a decrementing for loop until we get to 0. The revString is going to add to itself on every iteration of this loop since the loop is using the index and running backwards. We will get the last character added to the revString and then the next to the last until we hit zero which will be the first character. Then we will return revString back to our controller function.

The function displayString(revString) is our view function. It allows us to take the revString and display it. We use the template literal to display the message in the alert box instead of using + to concatenate the string. It allows for cleaner code to concatenate strings. It turns on the alert box by removing invisible by using classList and this allows the results to be displayed. The revString is being passed on and invisible is removed so the box shows up with the reversed string.
